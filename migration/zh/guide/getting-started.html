# 参数设置

  • 每批次查询条数：如果目标mysql没有设置过max_allow_packet ，默认为1m, 则此项不能设置过大，要考虑实际字段长度
  • 多线程使用：如果需要按顺序插入目标表，则不能打开多线程，当然，绝大多数情况都没有这样的需求。
