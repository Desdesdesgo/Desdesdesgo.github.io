# 介绍

# Migration 的需求产生于：
# 大量数据在迁移时的缓慢，特别是使用一些数据库工具的导出和导入
# 迁移中间处理往往需要通过编码实现
# 迁移的记录和任务的重复使用

# 系统构成

应用程序由electron 构成，界面由vue编写，迁移程序由java进行处理。

# 特性

  • 通过界面配置数据库连接
  • 界面交互选择数据表
  • 界面交互选择映射字段
  • 勾选配置项
  • 多线程支持
  • 分页查询设置
  • 中间脚本编写编辑器
  • 中间脚本测试
  • 实时迁移进度监控
  • 任务保存与编辑
